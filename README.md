yofi x kriya

# To Run this Project
- git clone https://github.com/yofiyonadio/kriya.git

# Run via Docker
- docker compose up

# Run Compile results
- npm install
- npm run build:prod

# Run via Nodemon
- npm install
- npm run serve

# Run via ts-node
- npm install
- npm run server

Server will run at http://localhost:3320/api/v1.0
